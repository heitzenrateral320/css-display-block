My Favorite Project

My favorite project from Web and App Development was the Space Invaders project last year where we programmed a spaceship to move left and right to shoot aliens dropping from the top of the screen. For this project I had to learn how to use Python and Pygame. I loved this project the most because it was the first full fledged game that I made and I really enjoyed making it.
